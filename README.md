# Information Security Risk Management – Jungle Kitchen

Comprehensive ISSRM project applying **OCTAVE Allegro** to a small food manufacturer.
Includes asset inventory, CIA evaluation, quantitative risk (SLE/ARO/ALE), mitigation roadmap,
and a security awareness session.

## Contents
- **/reports** – Final ISSRM report & risk analysis (PDF)
- **/slides** – Awareness deck: *Defend Together: Building a Cyber-Secure Workplace*
- **/artifacts** – Architecture diagram, threat trees, screenshots (Wazuh, dashboards)
- **/templates** – Risk model spreadsheet (sanitized)

## Highlights
- Asset-based risk assessment (Google Drive, Gmail, BYOD, cloud backups)
- CIA analysis + quantitative risk (SLE/ARO/ALE)
- Mitigations: MFA, encryption, RBAC, backup diversification, IR plan, awareness
- Bonus: Wazuh SIEM lab screenshots (install, dashboard, agents)
